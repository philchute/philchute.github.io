Enemy Territory was developed by Spash Damage as a Return to Castle Wolfenstien multiplayer expansion, but did not receive a retail release and was released as freeware in 2003, and later the code was released as open source.
The World War 2 gameplay features five classes: medic, engineer, soldier, field ops, and covert ops.
Enemy Territory: Legacy is a fully compatible client and server update to Enemy Territory.
Enemy Territory maintains active pubs with 22v22 gameplay nightly, also pickups, scrims, and tournaments organized on discord.
